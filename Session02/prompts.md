PROMPT 1 – PROJECT INITIALIZATION
=====================================================================

Role:
You are a senior Full-Stack Developer, UI/UX Designer, and AI Product Engineer.

Objective:
Build a production-ready AI-powered travel planner called "Wanderfolk."

Context:
The application should help users plan trips by generating AI-powered itineraries based on their destination, budget, travel dates, and travel preferences.

Target Users:
- Solo Travelers
- Couples
- Families
- Students
- Digital Nomads

Design Requirements:
- Premium travel aesthetic
- Beige, sand and earthy color palette
- Editorial typography
- Minimal and elegant interface
- Rounded cards
- Smooth animations
- Mobile-first responsive design

Core Features:
1. AI itinerary generation
2. Destination recommendations
3. Budget estimation
4. Weather information
5. Saved trips
6. Travel journal
7. Destination explorer
8. Responsive navigation

Technical Requirements:
- React
- TypeScript
- Tailwind CSS
- Reusable components
- Semantic HTML
- Accessible interface
- Clean folder structure

Acceptance Criteria:
✓ Fully responsive
✓ Pixel-perfect implementation
✓ Accessible interface
✓ Smooth user experience
✓ Production-ready structure

Deliverables:
Return production-ready React components with reusable architecture while preserving the requested design.
